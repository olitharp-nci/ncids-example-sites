# simple-ncids-app

Create a simple webpack app using NCIDS

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

## Running

```sh
npm run start
```

## Testing

To run unit tests:

```sh
npm test
```
